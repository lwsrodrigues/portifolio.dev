Sistema IOT (Internet das coisas)

Desenvolvi um projeto IoT utilizando o simulador Wokwi, no qual montei um circuito com um ESP32 e um sensor de umidade e temperatura (DHT). Para o envio e recebimento de dados em tempo real, criei um canal no ThingSpeak, configurando-o para receber informações via protocolo MQTT e a programação foi realizada na linguagem C.

Desenvolvimento Mobile + integração IoT

Desenvolvi um aplicativo Mobile utilizando a instalação do Apache Cordova no ambiente de desenvolvimento, juntamente com o Android SDK para compilar e testar o aplicativo no Android. Para gerenciar as dependências e automatizar a construção, foi utilizado Gradle. A instalação e configuração do Node.js foram essenciais para permitir o uso do Cordova CLI e facilitar a criação de pacotes para as plataformas móveis. Também foram feitas algumas alterações no JDK para garantir a compatibilidade. Durante o desenvolvimento, descobri que, além do Android, é possível integrar o aplicativo para iOS simplesmente adicionando a pasta apropriada ao projeto.
